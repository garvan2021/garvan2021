## 工具/软件

- [nmap](https://nmap.org/)
超好用本地端口检测工具

Ubuntu安装：
```shell
sudo apt install nmap
```

扫描本机端口：
```shell
sudo nmap localhost
```

### 变量替换
- 不使用变量替换
```shell
cd /home/jawen
```

```shell
cd $home
```
